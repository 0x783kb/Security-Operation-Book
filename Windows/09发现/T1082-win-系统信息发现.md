# T1082-Win-系统信息发现

## 来自ATT&CK的描述

攻击者可能试图获取有关操作系统和硬件的详细信息，包括版本，补丁，修补程序，服务包和目录结构。攻击者可以在发现过程中使用系统信息中发现的信息来决定后续的操作，包括攻击者是否完全感染了目标或尝试了特定操作。

## 测试案例

### windows

获得此信息例如命令和实用程序包括VER，SYSTEMINFO，dir用于获取基于本文件和目录的信息。

## 测试留痕
